松鹤·智语 演示页 — 变成手机能打开的 https 网址
================================================

本文件夹里的 index.html 为完整单页：
- 样式：同目录 songhe-built.css（预构建 Tailwind，不依赖境外 tailwindcss.com）
- 脚本：Vue / Vue Router 使用国内 BootCDN（cdn.bootcdn.net）

修改页面后请先在本目录执行：npm run build:css
再重新部署（如 npx vercel deploy --prod）。

方法一：Netlify Drop（最快，约 1 分钟，免费）
----------------------------------------------
1. 电脑浏览器打开：https://app.netlify.com/drop
2. 把整个「songhe-online」文件夹拖到网页里（或打成 zip 再拖入）
3. 等待几秒，会得到类似：https://随机名.netlify.app
4. 用手机浏览器打开该地址即可（需手机能上网）

方法二：Vercel
----------------------------------------------
1. 打开 https://vercel.com 注册/登录
2. New Project → Import，上传本文件夹或连接 Git 仓库
3. 部署完成后使用分配的 .vercel.app 域名

方法三：仍用家里电脑当服务器（仅同一 WiFi）
----------------------------------------------
在上一级目录双击「手机打开松鹤演示.cmd」，手机访问：
http://你电脑的局域网IP:8765/
（注意：电脑要一直开着，且与手机同一 WiFi）

更新页面内容后
----------------------------------------------
不要直接用资源管理器复制「松鹤智语-移动端演示.html」改名为 index.html（易损坏 UTF-8，页面会白屏）。
请在本目录 PowerShell 执行：.\sync-from-demo.ps1
然后：npm run build:css
再在 Netlify/Vercel 上重新部署或重新拖一次 Drop。
